Immersive Map on Flights Hub Page [Shipped]

Idea is to have an interactive map component on the flights hub page to make the page more user friendly and improve the enagement. This feature motivates the user to unlock and explore the immersive experience to help fnd the best and cheapest flights to favourite countries, continets or anywhere. During the flight we have noticed great engagement as Dwell time(75th %ile) on the flights hub increased 20% and funnel conversion from hub to listing gone up from 43% to 63% and total bookings from hub has nearly doubled. All this acheived without any performance hit keeping the metrics like PLT, LCP and FCP nearly the same.

Health Expert Platform Rewards [Shipped]

We have successfully integrated Microsoft Rewards into the Health Expert Creator Platform. In this phase, we have allocated 500 reward points for each question answered by expert. After an expert provides an answer, it undergoes a verification process. Once verified, the user can view the earned reward points on their rewards dashboard.
Experts have the opportunity to earn up to 5,000 reward points daily by answering up to 10 user asked questions. While experts are welcome to answer more than 10 user asked questions, they will not receive additional rewards beyond the daily limit. This initiative aims to incentivize experts to actively respond to user questions, enhancing engagement and support on the platform.
