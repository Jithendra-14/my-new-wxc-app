Relevance - People Also Ask

People Also Ask (PAA) Chat entry - Shipped

The new feature enables users to engage with Bing search's chat mode, a conversational interface designed to offer enhanced information and assistance. When a question is chosen from the PAA section, users will be seamlessly transitioned to the chat canvas, where they can input text and view chat mode messages. The chosen question serves as the initial context for the chat mode, allowing it to anticipate the user's interests and deliver pertinent responses. This functionality streamlines user access to the chat mode, facilitating quicker access to additional information and assistance within Bing search.
This feature has achieved a Chat DAU of 900k with a peak of 1.13M..

Relevance - Auto Suggest

Multiple Suggestions in Bing Chat - PoC

The Multiple suggestions within Bing Chat feature is designed to enhance the chances of aligning with the user's intended query by presenting multiple suggestions. This functionality will be accessible via both keyboard and mouse inputs. Upon typing, a pane will display multiple suggestions for the user. They can navigate through these suggestions using the Arrow or Tab keys and select a suggestion by pressing Enter.

Bing Sports

Redesigning and Migrating Sports Answers

We have been part of redesigning and migrating the team sports answered powered on Bing to the SnR framework from Budlite framework. As DE we have been actively looking to create an experience with a major focus on prominence styles, dark mode support, mobile responsiveness which were missing earlier.
In the last quarter, the team has flighted and shipped Soccer team sport on Bing with significantly high impact. One of the answer scenarios raised APSAT close to 87% after the redesign, earlier it was around 55%.

We have introduced a team sports answer branding with a fancy header layout built only using CSS styling, without impacting Page perf and load time, and is helping us catch better visibility.

Impact on reduction on JS client error by -5.61% (p-value = 5.99E-05) and Non JS client error -13.59% (p-value = 0) was also observed on the flight. Along with this a major reduction in Overall app size with lightweight HTML and SASS styling also was observed with the introduction of SML answer on bing desktop and mobile. HTML Response Size@90 by -5370 and HTML Response Size@Avg by -4121.7466.

These helped the Soccer team sports flight to get faster shipping approval and with that, now we have also powered other team sports answers like Baseball, Basketball, Football, Rugby, and more, which are currently under flight.

Project Phoenix

Sidepane Animations - Shipped

With a focus on improving the experience, we've introduced captivating native Sidepane animations. Since this is a legacy pane, animating it meant that we couldn't use the modern C++ APIs provided by Chromium to animate it. We had to add animations by adding an extra layer on top, animating the added layer by modifying its position, and then replacing the layer with pane content.

Health

SVG migration on Peregrine - Shipped

As an integral part of an ongoing initiative to optimize the code bundle size within the Peregrine framework, we took on the responsibility of migrating the SVG icons associated with Health experiences to CDN URLs. This migration encompassed the seamless transition of all relevant SVGs for both light and dark modes.
The results of this migration have been truly impactful. we've managed to achieve a substantial reduction in the overall bundle size, shedding a total of 940.5KB. This accomplishment not only enhances the efficiency and performance of our framework but also contributes to a more streamlined and optimized user experience for our users. It's a significant step forward in our efforts to ensure the framework's responsiveness and overall effectiveness.
